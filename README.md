# 🤖 AI Chat Application

A simple and modern AI-powered chat application built as a mini project 
using plain HTML, CSS, and JavaScript — no frameworks needed!

## ✨ Features
- 🔀 Supports multiple AI providers (Claude & Gemini)
- 💬 Real-time AI responses
- 🧠 Remembers full conversation context
- ⌨️ Typing indicator while AI is thinking
- 🗑️ Clear chat button
- 📱 Mobile responsive design
- ⚡ No installation required — just open in browser

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Anthropic Claude API
- Google Gemini API

## 🚀 How to Use
1. Clone or download the repository
2. Open `index.html` in any browser
3. Select your AI provider (Claude or Gemini)
4. Paste your API key
5. Start chatting!

## 🔑 Get API Keys
- **Anthropic (Claude):** https://console.anthropic.com
- **Google (Gemini):** https://aistudio.google.com

## 📁 File Structure
chat-app/
├── index.html   → App structure
├── style.css    → Styling & themes
└── script.js    → Logic & API calls

## ⚠️ Note
API keys are entered at runtime by the user and are 
never stored in the code — safe to use publicly.
